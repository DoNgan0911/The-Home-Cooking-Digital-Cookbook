# The-Home-Cooking-Digital-Cookbook
A digital curation repository created for INFO 4730 Final Project

This repository contains a curated collection of 5 digital items documenting
Vietnamese cultural food traditions. Each item includes structured metadata for preservation and
long-term accessibility.

## Items
- [Pho](items/pho.md)
- [Vermicelli Beef Soup](items/bunbohue.md)
- [Fried Rice](items/friedrice.md)
- [Chocolate chip biscuit](items/biscuits.md)
- [Spaghetti](items/spaghetti.md)

Metadata for each item is stored in the `metadata` folder.

## Access
All files are publicly accessible and preserved in Markdown format.
